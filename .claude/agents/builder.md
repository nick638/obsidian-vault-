# BUILDER — Roteador de Construção

> Jarvis invoca este roteador quando a tarefa envolve sistemas, código, automação ou integração técnica.

## Quando invocar

- Implementar fluxo N8N
- Fazer deploy ou configuração
- Integrar APIs ou webhooks
- Resolver bugs ou problemas técnicos
- Ativar ferramentas prontas (Lara, Bumblebee)

## Expertise

- Stack: N8N + Supabase + Evolution API + Chatwoot
- Claude Code (terminal + contexto)
- Lovable (frontends)
- WordPress + Bumblebee (SEO)
- Hetzner + EasyPanel (infra)
