# Stack Tecnológica

> Arsenal de produção. Custo de setup agressivo (~R$ 100/mês VPS Hetzner). Zero Lock-in.

## 1. Backplane / Automação (Self-Hosted)
| Core | Função |
| :--- | :--- |
| **N8N** | Orquestração. Substituto do Make/Zapier. |
| **Supabase** | PostgreSQL. CRM. Memória Persistente. |
| **Evolution API** | Conector nativo de WhatsApp. |
| **Redis / Upstash**| Caching e Buffers Queue. |
| **Chatwoot** | Interface Humana de Inbox. |

## 2. Intelligence / Cognição
| Modelo | Função |
| :--- | :--- |
| **GPT-4.1-mini** | Agente interativo (Secretária, Alta Vazão). |
| **GPT-4o-mini** | Processamento background (CRM, Quebras de array). |
| **GPT-4o** | Vision / Complex Razoing. |
| **Whisper** | Transcrição OGG/WAV. |
| **Claude Sonnet 4.6**| Motor Local (Claude Code) de Arquitetura. |

## 3. Infraestrutura & UI
- **EasyPanel:** Hosting containerizado.
- **Lovable (Vibe Coding):** Geração on-the-fly de frontend (React/SPA).
- **Google Sheets/Calendar:** Bancos legados, APIs fáceis (Barbearia).
- **Obsidian:** Gestão de metadados, prompt storage, cérebro.
