# GUIA DE EXECUCAO NO CANVA PRO

## Setup inicial

1. Criar documento 1080x1080 com fundo sólido #4A2E80
2. Importar Playfair Display via Adicionar fonte (disponível em Google Fonts no Canva Pro)
3. Inter já disponível nativamente no Canva
4. Criar paleta de marca personalizada: #4A2E80 / #FF6B35 / #D4AF37 / #FFFFFF / #D9C8F5

## Badge dourado (elemento reutilizável)

- Shape: retângulo bordas arredondadas (radius 20px)
- Preenchimento: #D4AF37
- Texto: Inter SemiBold, #4A2E80, uppercase, letter-spacing 1px
- Sombra suave: 0px 2px 8px rgba(0,0,0,0.2)
- Salvar como elemento de marca — reutilizar nos 3 formatos e 3 variações

## Overlay da foto

- Opção 1: retângulo roxo #4A2E80 sobreposto à foto, opacidade ajustada (0% centro, 75% bordas) com efeito Fade do Canva
- Opção 2: usar Ajuste > Desfoque de fundo + Fade nas bordas
- Resultado esperado: foto integrada ao fundo roxo sem linha de corte visível

## CTA button

- Retângulo #FF6B35, border-radius 8px
- Padding: 16px vertical, 32px horizontal (ou full-width no story)
- Texto: Inter ExtraBold, uppercase, #FFFFFF, letter-spacing 2px
- Seta "→" imediatamente após o texto
