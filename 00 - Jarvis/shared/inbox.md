# Inbox — OpenClaw → Escritório Local

## Como usar (OpenClaw)

```

TO: jarvis-ceo | pesquisador | executor
TASK: <descrição clara da tarefa>
CONTEXT: <contexto necessário>
PRIORITY: alta | normal | baixa
CREATED: YYYY-MM-DD HH:MM
```
