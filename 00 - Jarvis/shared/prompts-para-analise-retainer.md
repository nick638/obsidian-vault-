# Prompts para Análise do Retainer Holos

> Use esses prompts no Claude Code (com acesso ao vault) para obter uma análise objetiva sobre a proposta de retainer estratégico.

## Prompt 1 — ROI e Custo‑Benefício
```
Analise o custo‑benefício de um retainer de R$ 2.500/mês para Holos, considerando:
- Dados de mercado: pacote equivalente custa R$ 8.000–21.000/mês.
- ROI projetado: até R$ 114.000/ano de receita adicional.
- Liberação de 16h/mês de trabalho manual.
- Payback: 1 venda/mês cobre o módulo de conteúdo.

Perguntas diretas:
1. O investimento é justificável?
2. Qual o risco de não formalizar?
3. Recomenda aceitar, negociar ou rejeitar?
```

## Prompt 2 — Comparação com Mercado
```
Compare a proposta de retainer interno (R$ 2.500/mês) com contratação externa (R$ 8k–21k/mês). Liste:
- Vantagens e desvantagens de cada opção.
- Impacto no alinhamento estratégico.
- Custos ocultos (tempo de onboarding, risco de qualidade).
```

## Prompt 3 — Reflexão sobre Valor Justo
```
Com base nos dados do vault, responda:
- Qual o valor justo a ser pago a um CTO de plantão que gera R$ 114k/ano de receita adicional?
- Como balancear desconto familiar com valor de mercado?
- O que é mais caro: pagar o retainer ou perder oportunidades por falta de suporte técnico?
```

## Prompt 4 — Análise de Risco
```
Avalie os riscos de NÃO formalizar a relação com o Nicolas:
- Risco operacional (interrupções, dependência de boa vontade).
- Risco estratégico (desalinhamento, perda de oportunidades).
- Risco relacional (frustração acumulada).

Recomende uma mitigação.
```

## Prompt 5 — Tomada de Decisão (para reunião)
```
Prepare um resumo de 1 minuto para defender a proposta de retainer, incluindo:
- Dados de mercado (comparação).
- ROI claro.
- SLA e benefícios operacionais.
- Por que o desconto família ainda é vantajoso para o negócio.
```
