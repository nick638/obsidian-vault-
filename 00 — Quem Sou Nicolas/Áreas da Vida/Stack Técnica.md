# Stack Técnica

## Propósito
Ferramentas que compõem o arsenal de Nicholas. Dividido entre o que ele opera e o que o ecossistema (Jarvis) opera.

---

## Nicholas opera
- **IA:** Claude Code · Pixel Agents · Typebot
- **Automação:** N8N · Evolution API
- **Atendimento:** Chatwoot
- **Banco de dados:** Supabase
- **Infra:** Hetzner · EasyPanel · Cloudflare
- **Site:** WordPress · Elementor
- **Comunicação:** Telegram (OpenClaw)

## Ecossistema opera (Jarvis + agentes)
- **Frontend:** Next.js · React · Tailwind
- **Versionamento:** GitHub
- **Escritório local:** Claude Code (VPS: OpenClaw)
- **Mídia:** CapCut · Kling AI · Remotion

## Fluxo principal
```
Nicholas (Telegram) → Jarvis (VPS/OpenClaw) → GitHub → Escritório Local (Claude Code) → Entrega
```
