# Stack Vibe Coding — Rafael Melgaço

> "O terminal é o habitat natural da IA."

## 1. O Trio Base
- **Claude Code:** Motor executor (Assinatura Pro, nunca API/Token).
- **GitHub (gh CLI):** Versionamento, backup, auto-commit.
- **Obsidian:** Memória persistente. Claude lê o que você escreveu uma vez e lembra para sempre.

## 2. Hierarquia de Orquestração
- **Plugins:** Growth OS (9 agents + 20 skills).
- **Agents:** Entidades com contexto próprio.
- **Skills:** Manuais de especialidade (/copy-master, etc).
- **Agent Waves:** Execução paralela (Exclusivo do Terminal).

## 3. Aplicação Jarvis
- Rodar Claude Code via terminal (WSL). Libera Waves de agentes.
- Instalar `Claude Octopus` (Roteamento de modelos Sonnet/Haiku).
- Transformar Squads em plugin versionado no GitHub.

---
- **Auditoria:** Arquitetura do stack, Agent Waves e Claude Octopus preservados. Caveman style.
