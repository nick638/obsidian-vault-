# Holos — Site 2.0

> Redesign completo (WordPress + Elementor Pro). Foco em conversão por avatar e SEO orgânico.

- **Arsenal Técnico:** Parallax, Video Loop (Hero), Filtros AJAX (JetSmartFilters), ACF Pro, Lottie Icons.

## 2. Estrutura de Conversão (Avatares)
O site é segmentado por 3 caminhos claros:
1. **Quero Nova Carreira:** Foco em autonomia e formação completa (1200h).
2. **Quero Especializar:** Foco em novas técnicas para terapeutas (+65 cursos).
3. **Bem-estar Pessoal:** Foco em autocuidado (Reiki, Chakras).

## 3. Integração AI (N8N → WordPress)
Camada de atualização automática via WP REST API + ACF Pro.
- **Atualizar Preços:** N8N → ACF Field (Ex: Auriculoterapia R$ 680).
- **Turmas:** Criação dinâmica de posts/CPT.
- **SEO (Bumblebee):** Publicação 2 posts/dia via REST API.
