# Claude Code + Nano Banana Pro + WP Custom Theme

> Fluxo para sites de R$ 10.000.

## 1. Componentes
- **Claude Code:** Agent-teams escrevendo HTML/CSS/JS "Production Ready".
- **Nano Banana Pro (Gemini 3 Pro):** Geração de imagens custom fotorrealistas e consistentes.
- **WP Custom Theme:** Criar tema do zero (sem Elementor) para performance máxima.

## 2. Técnicas Premium
- **GSAP + ScrollTrigger:** Animações sofisticadas (Substituir AOS).
- **Locomotive Scroll:** Smooth scroll (~9kb).
- **3D Mouse-tracking:** Elementos seguem o cursor.

---
- **Auditoria:** Referências técnicas (GSAP, Locomotive Scroll) e Nano Banana Pro preservados. Caveman style.
