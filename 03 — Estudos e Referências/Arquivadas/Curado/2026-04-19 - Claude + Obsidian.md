# Claude + Obsidian — Mapa de Integração

> 4 caminhos para conectar Claude ao Vault.

## 1. Caminhos de Integração
1. **Acesso Direto (Path):** Claude Code lê/escreve via terminal. Sem busca semântica.
2. **MCP via Local REST API (Recomendado):** Plugin `Local REST API` (Porta 27123) + Bunx. Permite busca full-text e metadados.
3. **Plugin Sidebar (Claudian/Cortex):** Claude Code roda dentro do Obsidian. Organização autônoma.
4. **Obsidian CLI (Nativo 2026):** Controla UI do Obsidian via terminal.

## 2. Aplicação Jarvis
- Usar **MCP via Local REST API** para busca semântica.
- Claudian para organização em massa da estrutura 00-04.
