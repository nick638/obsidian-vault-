# Primeiro Projeto com Vibe Coding — Rafael Melgaço

> Foco: Tino AI. Briefing → Planejamento → Execução.

## 1. Framework dos 3 Tempos
1. **Briefing:** Prompt sujo + "Me diga o que entendeu". IA interpreta sem gastar token de execução.
2. **Critério de Sucesso:** "Explique o que seria um resultado bem-sucedido". Define o fim antes do meio.
3. **Planejamento/Execução:** Aprova spec → Aciona `Epic Executor` (Skill).

## 2. Filtro do Problema
`Problema → Impacto → Pagariam? → Critério de Sucesso → Engenharia Reversa`.
Nunca criar o que já existe, exceto para inovar em custo ou velocidade.

## 3. Aplicação Jarvis
- Usar Framework 3 Tempos em todo prompt do Claude Code.
- Definir sucesso da Lara antes de polir mais.
- Filtro de problema elimina 90% das ideias de dopamina.

---
- **Auditoria:** Framework 3 Tempos, Filtro de Problema e Tino AI preservados. Caveman style.
