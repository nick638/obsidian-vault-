# [PONTE] Estratégia de Captação

> **Presença cria visibilidade. Prospecção cria oportunidades. Sem os dois juntos, a renda depende de sorte.**

Visibilidade → [[Arquivo não encontrado — ver conteúdo migrado]] · Ação → [[Projetos/Clientes Externos/Prospecção Ativa]] · Destino → [[00 — Quem Sou Nicolas/Areas/Carreira]]

## Por que os dois precisam coexistir

Presença digital cria o contexto para a prospecção funcionar:
- O prospect pesquisa Nicolas antes de responder → encontra portfólio + cases → credibilidade antes da reunião
- A abordagem deixa de ser fria → "já ouvi falar de você"

Prospecção ativa acelera o que a presença faz devagar:
- Presença orgânica leva 3–6 meses para gerar leads
- Prospecção gera conversas em dias
- Cada cliente fechado → vira case → alimenta a presença

```
Presença Digital (conteúdo + autoridade)
        ↓                         ↑
        ↓        ciclo            |
        ↓                         |
Prospecção Ativa → cliente → case → conteúdo
```

## O paradoxo que trava Nicolas

Esperar presença perfeita para prospectar. Esperar cases para construir presença. Loop que mantém os dois parados.

Saída: fazer os dois em paralelo. O primeiro cliente não precisa de presença perfeita. A primeira presença não precisa de case perfeito.
