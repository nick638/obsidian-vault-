# Automação

> **O produto que Nicolas vende e o meio pelo qual vive.** IA + N8N + WhatsApp + Supabase = sistema de vendas autônomo que opera 24/7 sem contratar ninguém. O mercado cobra R$30–40k por sistemas que Nicolas entrega em 2 dias. A janela: 2024–2026 ainda é especialidade. Em 2027 vira commodity.

---

## Como tudo se conecta

```
Ideia de sistema
      ↓
Vibe Coding (Lovable) → cria a interface/sistema visual
      ↓
N8N → orquestra os fluxos e a lógica
      ↓
IA (OpenAI/Claude) → inteligência e conversação
      ↓
Evolution API → entrega no WhatsApp
      ↓
Supabase → armazena tudo no CRM
      ↓
Sistema rodando e pagando
```

---

## Por que essa combinação é poderosa

- **Vibe Coding** cria o frontend/sistema em horas sem código manual
- **N8N** conecta tudo sem precisar de backend complexo
- **IA** dá inteligência real ao sistema — não é só automação, é conversação
- **Evolution API** leva tudo pro WhatsApp — canal com maior conversão do Brasil
- **Supabase** guarda histórico, CRM, memória dos agentes

> Juntos, esses 5 elementos formam um sistema completo de vendas e atendimento que opera 24/7 sem equipe

## Arquitetura padrão dos sistemas

```
WhatsApp → N8N → Agente IA → Supabase (CRM)
                     ↓
              Subagente de conhecimento
                     ↓
              Follow-up automático
```

---

## Padrões que uso

- Buffer com Redis antes de processar mensagem
- Memória via PostgreSQL + LangChain
- Quebra de mensagens longas em partes menores
- CRM separado do agente principal
- Prompts em JSON para melhor estrutura

---

## O que estou aprendendo

- Prompts em JSON melhoram qualidade da IA
- Claude cria dashboards interativos com código
- Fluxo de aquecimento de lead no WhatsApp vale R$30k no mercado
- Clone de IA de especialista precisa: falas, padrões linguísticos, personalidade, tom
- Fórmula de oferta irresistível: (resultado × probabilidade) ÷ (tempo × esforço)
- Vibe Coding + N8N + IA = MVP em 2 dias

---

## Por que isso importa — automação é o único produto que escala sem contratar

O mercado cobra R$30–40k por sistemas que Nicolas entrega em 2 dias. Isso não é exagero de Leo Soares — é porque quem não sabe fazer não consegue sequer avaliar o custo real.

Mas o gap não é só técnico. É de percepção: a maioria dos donos de negócio acredita que automação = chatbot = coisa simples. O que Nicolas entrega é diferente: **sistema de vendas autônomo** — qualificação, CRM, follow-up, memória de conversa, integração com WhatsApp. Isso opera 24/7 sem contratar.

Para uma escola como a Holos: 5.768 contatos no ZenPro. Cada disparo → Lara qualifica → CRM registra → follow-up 3 vezes. Uma pessoa faria isso em como tempo? Um mês de trabalho manual = 1 hora com o sistema rodando.

**A janela:** 2024–2026 é quando isso ainda é especialidade. Em 2027–2028 vai ser commodity. Quem construiu autoridade antes disso tem 2–3 anos de vantagem de posicionamento.
