# Arquétipo — Nicholas Maier

> Quem Nicholas é quando está no seu melhor. Não aspiração — realidade observada.

## 1. Quem é
Nicholas vê o sistema por baixo antes dos outros. Acha a brecha. Resolve na raiz — não tapa buraco.

Quando está em hiperfoco, some do mundo. Perde noção do tempo. Fica irritado com interrupção — não é grosseria, é proteção do recurso mais escasso: atenção concentrada.

Constrói pra si primeiro. Quando está bom, replica pros outros sem depender de ninguém.

O que quer que digam sobre ele: *"Se não fosse o Nicholas, a gente tinha se lascado."*

## 2. Motor
Decifrar como algo funciona → construir uma versão própria → não depender de ninguém pra manter rodando. Por que esse motor existe → [[00 - Pessoal/Areas/Propósito]] · Como manifesta no trabalho → [[00 - Pessoal/Perfil]]

## 3. Combustível
Reconhecimento. Não é vaidade — é biologia de Projetor. Quando o trabalho é visto e valorizado, opera no máximo. Quando é ignorado, desliga. → [[00 - Pessoal/Diagnóstico Real]]

## 4. Inimigo
Preguiça intelectual — em si mesmo e nos outros. Quem não quer aprender o básico, Nicholas não ajuda.
