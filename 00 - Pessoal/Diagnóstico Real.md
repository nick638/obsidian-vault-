# Diagnóstico Real — Nicholas Maier

> Como o cérebro do Nicholas funciona. Base para entender por que ele age como age.

## 1. Como Nicholas funciona
- **Projetor 6/2:** Bateria limitada. Absorve energia do ambiente. Hiperfoco intenso quando engajado — some do mundo.
- **ENTP:** Pensa em sistemas. Ciclo de novidade de 5-7 dias. Dopamina na antecipação, não na entrega.
- **7w8 Maverick:** Liberdade acima de tudo. Busca brechas. Controle = proteção.
- **Gate 42 Faltante:** Não sente biologicamente o fim. Projetos ficam 90% prontos. Jarvis é o Gate 42 externo. Impacto direto na monetização → [[00 - Pessoal/Areas/Carreira]]

## 2. O que isso significa na prática
- **Ambiente > Disciplina:** Ambiente pesado derruba performance sem motivo aparente. Mudar contexto destrava.
- **Sprints de 5-7 dias:** Unidade real de tempo. Planejamento mensal não funciona como método — OK como marco financeiro.
- **Comunicação técnica:** Sem perguntas emocionais. Linguagem de sistema.
- **Nenhuma regra é absoluta:** Flexibilidade é parte do sistema, não falha.
- **1 projeto por sprint:** IA acelerou execução mas não capacidade de processar. Projetos demais = estresse = abandona tudo.
- **Critério de conclusão:** Jarvis cobra o "feito" antes de abrir qualquer projeto novo.

## 3. Armadilhas recorrentes
- **Conteúdo novo = abandono:** Vê vídeo novo → larga o projeto atual → abre outro. Antídoto: Jarvis assume o braçal, Nicholas só decide.
- **Ambiente como bateria:** Performance cai quando ambiente está pesado, mesmo sem perceber.
- **Hiperfoco com prazo:** Engajamento dura 5-7 dias. No dia 8 a novidade acabou. Solução: criar novidade artificial dentro do projeto ou encerrar antes do dia 8.
