## Como navegar

Comece por [[_INDEX]]. É o mapa vivo deste vault.

Se for a sua primeira vez aqui, leia nesta ordem:

1. [[_INDEX]] — o que tem no vault agora
2. [[00 - Pessoal/Perfil]] — quem é o dono
3. [[CLAUDE]] — a doutrina (regras que toda skill de IA segue)

## Como adicionar notas

Abra um template dentro de `_templates/` da área apropriada e duplique. Ou deixe uma
skill de IA escrever por você — ela vai seguir a doutrina automaticamente.

Se editar à mão e quiser que o `_INDEX.md` reflita suas mudanças, rode
`/obsidian-master-kit:sync` no Claude Code.
